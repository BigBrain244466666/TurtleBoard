# TurtleBoard

Description: A cute basic board in the shape of a turtle that lights up in the dark and buzzes when provoked!

BOM:
- 6x LED
- 1x Resistor
- 1x Photoresistor
- 1x Pushbutton
- 1x Mini motor disk
